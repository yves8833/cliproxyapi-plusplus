# Research

## Repository Fit

`cliproxyapi-plusplus` describes itself as an agent-native, multi-provider
OpenAI-compatible proxy. The README lists provider routing for OpenAI, Anthropic,
Gemini, Kiro, Copilot, and related model backends, which makes it a direct fit for
the sladge rollout criteria.

primary checkout on `main`. This rollout used `.worktrees/sladge-badge`.

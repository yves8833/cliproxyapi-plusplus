# Specifications

## Acceptance Criteria

- `README.md` includes exactly one `sladge.net` badge reference near the existing
  top-of-file badge block.
- The change is documentation-only.
- The canonical checkout remains untouched.
- The commit includes the required Codex co-author trailer.

## Assumptions, Risks, Uncertainties

- Assumption: A model-routing proxy belongs in the LLM-heavy rollout set.
- Risk: Badge proliferation could dilute meaning if applied to non-LLM repos.
- Mitigation: Keep rollout tied to the projects-landing governance ledger.
